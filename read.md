# 🛡️ Sentinel-Log SIEM
Gerçek zamanlı log analizi ve anomali tespit motoru.

### 🚀 Özellikler
* **Canlı Analiz:** Python Generator yapısı ile sıfır gecikmeli log okuma.
* **Tehdit Tespiti:** SQL Injection, Path Traversal ve Brute Force saldırılarını anında yakalar.
* **Görsel Dashboard:** Flask ve SSE (Server-Sent Events) ile sayfayı yenilemeden canlı alarm takibi.

### 🛠️ Kurulum
1. `pip install flask`
2. `python siem_motoru.py`
3. Tarayıcıda `localhost:5001` adresine git.
